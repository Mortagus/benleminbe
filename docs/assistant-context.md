## Regles De Lecture

- Lire d'abord l'index du domaine concerné.
- Lire ensuite le document de vision ou la specification du sujet.
- Lire le document de regle metier si la question porte sur un comportement precis.
- Lire le suivi actif seulement si la tâche est encore en cours.
- Lire l'archive uniquement pour comprendre une décision passée.

## Regles D'Usage

- Ne pas confondre `editorial/` avec une source runtime.
- Ne pas confondre un index avec une reference métier.
- Ne pas supposer qu'un document en cours fait autorité.
- Preferer les fichiers au nom explicite plutot que les `README.md` de sous-dossiers.
