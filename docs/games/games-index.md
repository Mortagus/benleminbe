# Documentation Des Games

Ce dossier regroupe la documentation stable de l'univers public `Games` de `benlemin.be`.

## Rôle

`Games` est l'univers public dédié aux petits jeux maison en HTML, CSS et JavaScript vanilla. Il doit rester simple à étendre sans dépendre du `Lab`.

## Points D'entrée

- [Univers du site et navigation](../site-universes-and-navigation.md) : découpage global des univers et navigation.
- [Simon](simon.md) : référence technique du premier jeu de l'univers `Games`.

## Organisation

```text
docs/games/
├── games-index.md
└── simon.md
```

## Règle De Rangement

- Garder ici la documentation stable des jeux publics.
- Ajouter un document par jeu quand son comportement ou son architecture mérite une référence dédiée.
- Déplacer les notes de suivi provisoires dans `docs/en-cours/` si un chantier sur un jeu reste actif.
